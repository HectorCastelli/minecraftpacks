# HectorCastelli Bucket-less

A small Datapack that gives you alternative to the iron buckets.

These are meant to be used mainly on the early game, when Iron is still scarce, and are meant to remove the need to rush for iron, allowing you to enjoy the early game for longer.

## Features

- Adds a new item "Crude Bucket" that:
  - Can only hold water. Lava will destroy it, and you might burn your hands!
  - Leaks over time, so it's not a permanent solution
  - Has limited uses
- Adds a new item "Ceramic Bucket" that:
  - Can only hold water. Using it on lava will cook it into a regular flower pot.
  - Has a limited lifespan. When you take damage, it might break
